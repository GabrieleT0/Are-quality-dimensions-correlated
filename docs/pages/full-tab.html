RQ2 - Correlation matrices for RQ2 (table 2. in the paper, expanded for every tool)

The underlying correlation matrices, were generated considering metrics that are measured by at least two tools, so as to simplify comparison. The metrics were then combined linearly into dimensions, but only where the metrics in the dimension are the same for all 4 tools, otherwise the metric is left without aggregating, this is to have a fair comparison of dimensions across tools. The linear combination is performed using functions implemented in the utils.py script, which is part of the publicly available GitHub repository. These functions, specifically those starting with the prefix “merge” can be accessed at the following link: https://anonymous.4open.science/r/AreQualityDimensionsCorrelated/src/utils.py . There are different functions depending on the tool, this is because of the different nomenclature used for the metrics measured.

Only significant correlations among the 4 tools were reported in the paper, making a union. The complete table is given here, including correlations less than 0.5. Finally, complete correlation matrices are also provided, divided according to the tool under consideration.
